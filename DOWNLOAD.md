## ✅ Cách chạy game

1. **Tải file ZIP** ↑ từ bảng trên
2. **Giải nén** thư mục
3. **Chạy file:** `Cat n Grass.exe`
4. 🎮 **Chơi game!**

---

## 🖥️ Yêu cầu hệ thống

- **OS:** Windows 7 trở lên
- **RAM:** 2 GB tối thiểu
- **Disk:** 300 MB trống
- **Graphics:** DirectX 11 compatible

---

## 📂 Các file trong gói game

```
CATstronaut v1.1.0/
├── Cat n Grass.exe          (Game chính)
├── UnityPlayer.dll          (Runtime Unity)
├── UnityCrashHandler64.exe  (Xử lý lỗi)
├── dstorage.dll
├── dstoragecore.dll
├── Cat n Grass_Data/        (Dữ liệu game)
├── MonoBleedingEdge/        (Mono runtime)
└── D3D12/                   (Graphics)
```
